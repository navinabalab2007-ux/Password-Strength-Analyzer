
# Password Strength Analyzer 🔐

A Streamlit-based application that analyzes password strength and provides security recommendations to users.

## 📌 Project Overview

Password Strength Analyzer is a Python application that evaluates the strength of a password based on different security rules. It helps users create stronger and more secure passwords.

## 🚀 Features

- 🔍 Password strength analysis
- 📊 Password score calculation
- ✅ Checks uppercase and lowercase letters
- ✅ Checks numbers and special characters
- ⚠️ Detects weak passwords
- 💡 Provides security recommendations
- 🖥 Interactive Streamlit user interface

## 🛠 Technologies Used

- Python
- Streamlit
- Regular Expressions
